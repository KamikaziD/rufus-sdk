How it works

Each browser tab = an edge Pod.


Sovereign — highest S(Vc) score.
Executes relay transactions + owns leadership.


Relay — online but not leader.
Routes transactions through Sovereign.


Offline — SAF mode.
Queues transactions locally; flushes on restore.


Scoring: S = 0.5·C + 0.15·(1/H) + 0.25·U + 0.1·P
